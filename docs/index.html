FEATURES

核心功能

任务胶囊（Task Capsule）

跨端流转的独立任务单元，包含目标、风险等级、执行位置。手机创建 → 后端调度 → 电脑执行 → 结果回传。

设备池（Device Pool）

同账号下多端自动组建设备池，实时心跳保活，在线/离线状态同步，设备能力声明（CPU/GPU/内存）。

联邦节点（Federation）

多台电脑组成联邦，任务可跨节点调度。基于 mDNS 的局域网节点发现，节点间通过 WebSocket 同步状态。

风险分级与审批

任务胶囊分低/中/高/极高四级风险。高风险操作推送到手机端，一键批准或拒绝，确保安全。

3
回到电脑前

电脑端已接入 TRAE/CC，代码正在执行，实时日志可见

4
需要审批

高风险操作推送到手机，一键批准或拒绝

5
执行完成

结果同步到所有设备，胶囊归档，任务上下文可导出迁移
